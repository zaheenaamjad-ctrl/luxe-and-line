### BUG 5: PAGE SCROLL — START FROM TOP ON NAVIGATION

When a user clicks a link and navigates to a new page, the page renders starting from the bottom/middle instead of the top.

Fix this by adding a ScrollToTop component:

Create a file src/components/ScrollToTop.jsx:
```jsx
import { useEffect } from 'react';
import { useLocation } from 'react-router-dom';

export default function ScrollToTop() {
  const { pathname } = useLocation();
  useEffect(() => {
    window.scrollTo(0, 0);
  }, [pathname]);
  return null;
}
```

Then in your main App.jsx or Router, import and add <ScrollToTop /> as the first child inside <Router> or <BrowserRouter>, before your Routes.
